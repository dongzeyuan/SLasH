## 琢磨琢磨IDLE这个游戏吧

一个典型的战斗流程是怎么样呢？
1. 进入某张地图
2. 遇到怪物
3. 进入战斗状态
4. 攻击怪物
5. 计算血量
6. 玩家或怪物的血量<=0时战斗结束
7. 进入掉落计算，计算掉落什么装备
8. 战斗结束
